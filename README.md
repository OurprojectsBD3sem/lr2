# **Доска объявлений на Django**

**Описание проекта**

Веб-приложение "Доска объявлений" разработанное на Django в рамках лабораторной работы. Проект реализует полный цикл разработки веб-приложения от создания моделей до работы с шаблонами и статическими файлами.

**Основные возможности**

Создание, просмотр и управление объявлениями
Категоризация объявлений по рубрикам
Административная панель для управления контентом
Адаптивные шаблоны с наследованием
Статические файлы (CSS, изображения)
Формы для добавления новых объявлений

**Технологический стек**

Python 3.13
Django 5.2.8
SQLite (база данных по умолчанию)
HTML/CSS для фронтенда
